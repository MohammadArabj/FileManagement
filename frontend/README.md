# Frontend

در این مرحله یک صفحه‌ی استاتیک برای تست سریع API فراهم شده است:
- `index.html`: دکمه‌ای برای باز کردن مدال مدیریت فایل دارد.
- `script.js`: به Endpointهای بک‌اند متصل می‌شود (`apiBase` را در ابتدای فایل تغییر دهید).
- `styles.css`: استایل ساده برای مدال و کارت فایل.

برای اجرا، فایل را با یک سرور استاتیک باز کنید (مانند VS Code Live Server). این صفحه صرفاً برای اعتبارسنجی رفتار API است؛ می‌توانید بعداً آن را با یک کامپوننت Angular (نسخه 17 یا 20) جایگزین کنید.
